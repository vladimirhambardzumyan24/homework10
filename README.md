2 How to take arguments in sum function?

```js
function sum() {
  return args.reduce((sum, current) => {
    return sum + current;
  });
}
sum(1, 2);
sum(1, 2, 3);
```

3 How to pass arguments in sum function?

```js
function sum(x, y, z) {
```
